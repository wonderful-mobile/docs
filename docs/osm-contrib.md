# วิธีเพิ่มข้อมูลลงใน OpenStreetMap

#### ผ่านคอมพิวเตอร์ PC

- iD Editor สามารถใช้งานได้ที่[หน้าแก้ไขของเว็บไซต์ OpenStreetMap](https://www.openstreetmap.org/edit)
- JOSM สามารถดาวน์โหลดได้จาก[เว็บไซต์ JOSM](https://josm.openstreetmap.de/) หรือดาวน์โหลดจาก [Flathub](https://flathub.org/apps/org.openstreetmap.josm) สำหรับผู้ใช้ Linux

#### ผ่านสมาร์ทโฟน

- StreetComplete สามารถดาวน์โหลดได้จาก [Play Store](https://play.google.com/store/apps/details?id=de.westnordost.streetcomplete) และ [F-Droid](https://f-droid.org/en/packages/de.westnordost.streetcomplete/)
- Vespucci สามารถดาวน์โหลดได้จาก [Play Store](https://play.google.com/store/apps/details?id=de.blau.android) และ [F-Droid](https://f-droid.org/en/packages/de.blau.android/)
- Every Door สามารถดาวน์โหลดได้จาก [Play Store](https://play.google.com/store/apps/details?id=info.zverev.ilya.every_door), [F-Droid](https://f-droid.org/en/packages/info.zverev.ilya.every_door/) และ [iOS App Store](https://apps.apple.com/us/app/every-door/id1621945342)
