# Wonderful Kernel

Wonderful Kernels are custom kernels focusing on stability, feature, performance and few more that can be added.

## Features

- Compiled with -O3 optimizations using Clang and a few additional tweaks for improved performance
- Supports KernelSU Next
- Supports SuSFS for enhanced root concealment
- Removed Knox hypervisor (Samsung devices)
- Supports APatch: You can patch the boot image with APatch for rooting (similar to Magisk).
- More CPU governors: Powersave, Userspace, Ondemand, Conservative, Schedutil.

## Notice​
- Wonderful Kernel is under active development; additional features will be added in future updates.​
- This kernel is marked as beta due to limited features and ongoing testing.​
- Please reflash the stock boot image before flashing this kernel or updating to the latest version!​
- If you're using Edward KernelSU or ro2rw, please reflash the stock firmware before switching to this kernel.​
- Ensure you flash with a non-Magisk/unrooted boot image (stock boot image).​
Important Warning​
- If anyone asks you to make your vendor partition read-writeable, do not trust them!​ Loading untrusted modules can **brick your device**, and making the vendor partition read-writeable can cause issues such as camera/front camera malfunctions.​ Always research thoroughly before making any changes to your system!​
