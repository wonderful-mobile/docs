# Wonderful Mobile Docs

This is the repository for TechTransThai Wonderful Mobile Documentation, containing a public knowledge base related to the Wonderful Mobile Project.

## Running the project

```
git clone https://github.com/wonderful-mobile/docs.git
cd docs/
mkdocs serve
```

Then visit http://127.0.0.1:8000/ in your browser.
